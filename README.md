# Iris Çiçeği Sınıflandırma Projesi
Bu proje, **Scikit-learn** kütüphanesi kullanılarak klasik Iris (Zambak) veri seti üzerinde bir makine öğrenmesi modeli eğitir.

## Proje Adımları
- Veri setinin yüklenmesi ve incelenmesi.
- Verinin eğitim ve test olarak bölünmesi.
- **Lojistik Regresyon** algoritması ile modelin eğitilmesi.
- Başarı metriklerinin (Accuracy, Precision, Recall) hesaplanması.

## Özellikler
- Algoritma: Lojistik Regresyon
- Veri Seti: Scikit-learn Iris Dataset
- Kütüphaneler: Pandas, Scikit-learn

## Kullanılan Teknolojiler
- Python 3.x
- Scikit-learn
- Pandas
- Jupyter Notebook

## Nasıl Çalıştırılır?
`odev_iris.ipynb` dosyasını VS Code veya Jupyter Notebook üzerinden açarak tüm hücreleri sırayla çalıştırabilirsiniz.

## Sonuç
Model test verileri üzerinde %95+ doğruluk oranı sağlamıştır.
